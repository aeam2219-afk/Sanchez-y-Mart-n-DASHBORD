⚠ Notas sobre los datos
  • Stock = col[15] "Unidades disponibles aptas para la venta" — excluye unidades dañadas.
  • Junio tiene 4 semanas de datos: 1-7 Jun, 8-14 Jun, 15-21 Jun, 22-28 Jun.
  • El pronóstico usa Abr+May+Jun (80 días) como base de velocidad de venta.
  • Las devoluciones MXN son estimadas (uds × precio prom.). El monto real lo acredita Amazon.
  • Meta mensual fija: $1,000,000 MXN.
